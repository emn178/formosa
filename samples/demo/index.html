Date

jquery-ui

formosa uses jQuery datepicker for date input in default.

Source Code

          <input name="date" type="date" value="2016-01-01" 
          data-options='{"dateFormat": "yy-mm-dd"}' />

Result

bootstrap-datepicker

formosa also supports Bootstrap datepicker for date input. You can use data-vendor to specify.

Source Code

          <input name="date" type="date" value="2016-01-01" data-vendor="bootstrap-datepicker"
          data-options='{"format": "yyyy-mm-dd"}' />

Result

Browser Default

You can use data-norender if you want to use browser default input.

Source Code

          <input name="date" type="date" value="2016-01-01" data-norender />

Result

Datetime

bootstrap-datetimepicker

formosa uses Bootstrap datetimepicker for datetime input. You can set a default options for specific type.

Source Code

          formosa.types.datetime.options = {
            format: 'YYYY-MM-DD HH:mm'
          };
        
          <input name="datetime" type="datetime" value="2016-01-01T00:00:00+08:00" />

Result

Color

kolor-picker

formosa uses kolor-picker for color input. You can also set options by using JavaScript.

Source Code

          <input id="color" name="color" type="color" value="rgba(0,0,0,1)" />
          <span id="color-text"></span>
        
          $(document).ready(function () {
            $('#color').data('options', {
              onSelect: function (color) {
                $('#color-text').text(color);
              }
            });
          });

Result

International Telephone

intl-tel-input

intl-tel is custom input type. formosa uses intl-tel-input for this.

Source Code

          <input name="tel" type="intl-tel" value="+886987654321" />
